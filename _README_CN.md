# SQL 手册

一个全面的 Quarto 网站，提供 SQL、R 和 Python 常见数据操作任务的并行教程。这个资源帮助您理解这三种流行的数据分析工具之间的等效操作。

## 特性

- **语言对比**：并排示例展示如何实现相同的数据操作任务
- **全面覆盖**：从基础探索到连接和旋转等高级操作
- **交互式示例**：通过 Quarto 实时代码执行
- **双语支持**：提供英文和中文版本
- **现代库**：使用当前最佳实践，包括 DuckDB、polars、pandas 和 dplyr

## 内容

- 数据探索入门
- 基础查询（选择、过滤、排序）
- 数据转换（创建列、分组）
- 表操作（创建、更新、删除、丢弃）
- 高级操作（连接、联合、旋转/逆旋转）
- 美化表格显示

## 技术栈

- [Quarto](https://quarto.org/) 用于发布
- [DuckDB](https://duckdb.org/) 用于 SQL 示例
- [R](https://cran.r-project.org/) 配合 tidyverse 进行数据操作
- [Python](https://www.python.org/) 使用 pandas 和 polars
- [Babelquarto](https://github.com/paithiov909/babelquarto) 用于多语言支持
