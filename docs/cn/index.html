0.1 介绍数据分析工具

在数据分析中，我们使用不同的工具，每个工具都有其优势：

  • SQL：擅长结构化查询和数据库操作
  • R：强大的统计计算和数据可视化
  • Python：多功能编程，拥有优秀的数据科学库

本教程展示了如何在所有三个工具中完成相同的数据操作任务，帮助您为您的需求选择正确的方法。

# UNPIVOT (逆透视)

本节介绍如何将数据从宽格式转换为长格式。

::: panel-tabset

### SQL

```{sql}
#| connection: con
-- 对先前透视的表进行逆透视。
UNPIVOT
(
  PIVOT cities
  ON year
  USING sum(population)
  GROUP BY country
)
ON COLUMNS(* EXCLUDE (country))
INTO
    NAME year
    VALUE population;
```
